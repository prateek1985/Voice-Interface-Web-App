
Description
To run Annyang on localhost navigate to folder containing gruntfile.js and execute grunt dev in cmd

1)I have used Annyang API for Speech to text Conversion and using this voice commands are converted to http calls.
2)Navigate to index.html in project folder and run it on localhost as described above.
3)After opening index.html on localhost you can do the following-

a)In order to view a place on google maps just say the following command-
show me (name of the place)

b)In order to get  directions from one place to another say-
drive me from (source) to (destination)

c)In order to search nearby places such as restaurants,schools,cinema etc say-
find me (the phrase)

where phrase can be anything like "restaurant in London" etc. 

my main code is in demo/index.html
